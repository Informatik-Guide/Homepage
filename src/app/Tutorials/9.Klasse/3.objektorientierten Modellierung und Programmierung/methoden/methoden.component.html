Mathoden
Der Methodenkopf definiert den Namen, die Parameterliste und den Rückgabetyp einer Methode. Dieser steht am Anfang der Methode.
Der Methodenrumpf enthält die eigentlichen Anweisungen und den Code, der in einer Methode ausgeführt wird
Die Kombination von beidem wird Methodendefinition genannt, die das Verhalten der Methode definiert.
Der Methodenaufruf ist der Prozess, bei dem eine Methode aufgerufen wird, um bestimmte Aktionen auszuführen. Dabei gibt die Methode oft einen Wert zurück, der Rückgabewert genannt wird. Allerdings gibt es auch Methoden, die keinen Wert oder Meldung zurückgeben.
Ein Konstruktor ist eine spezielle Methode einer Klasse, die automatisch beim Erstellen eines Objekts aufgerufen wird. Diese bestimmt die Attributwerte der Klasse und erstellt das Objekt. Bsp in Java :

Geben
Diese Methode gibt den Wert eines Attributes. Dazu braucht sie immer einen Rückgabewert (hier: int).
Setzen
Beim Aufrufen dieser Methoden wird der Wert eines Attributes neu gesetzt. Dabei gibt es keinen Rückgabewert, deswegen wird hier „void“ benutzt.
Setzen und Geben
Bei dieser Methode werden bei Funktionen kombiniert.
For-Schleife
Bei der Methode wird eine bestimmte Anweisung, so oft ausgeführt wie die angegebene Bedingung erfüllt ist.
Als erstes eine Zählvariable festgelegt, die verwendet wird um den Fortschritt der Schleife zu verfolgen
Zweitens wird die Bedingung angegeben, die erfüllt sein soll, dass die Schleife ausgeführt wird.
Als letztes muss die Zählvariable aktualisiert werden, also einen neuen Wert annehmen. (Dies ist Alles in den Klammern bestimmt)
If-Else
Hier wird eine Bedingung bestimmt und wenn diese erfühlt wird, dann wird der Quellcode ausgeführt der in den geschweiften Klammern steht.
Wird die Bedingung nicht erfüllt wird automatisch in den Else-Zweig gesprungen.
Diese Anweisung kann auch verschachtelt werden.
While-Schleife
Diese Schleife führt den Code aus, solange eine angegebene Bedingung erfüllt ist (in runden Klammern)
Davor muss jedoch die Zählvariable festgelegt werden (hier int i=1) und danach muss diese auch aktualisiert werden (hier i++).
