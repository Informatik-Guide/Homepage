Start
Variablen sind Platzhalter, denen bestimmte Werte zugeordnet werden können. Es gibt verschiedene Arten von Variablen: Parameter, lokale Variable und Attribute.
Ein Parameter ist eine Variable, die an eine Methode übergeben werden, wenn sie aufgerufen wird. Parameter werden benutzt, wenn zur Ausführung der Methode noch zusätzliche Informationen benötigt sind.
Eine lokale Variable existiert nur in einer Methode und werden innerhalb dieser Methode deklariert. Diese Art von Variable kann dann auch nur in dieser einen Methode verwendet werden.
Attribute werden benutzt um Eigenschaften der Objekte festzulegen. Attribute sind globale Variablen, die in jeder Methode der Klasse verwendet und gelesen werden können.
Ein Übergabewert wird an eine Methode weitergegen, dass diese ihn als Parameter weiterverwenden kann.
Durch Wertzuweisung, wird der Wert einer Variable definiert oder der bisherige Wert geändert.
