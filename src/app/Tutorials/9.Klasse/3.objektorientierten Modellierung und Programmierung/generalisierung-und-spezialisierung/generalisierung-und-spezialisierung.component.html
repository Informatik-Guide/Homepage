Vererbung
Eine Oberklasse ist eine abstrakte Klasse, von der andere Klassen erben können, während eine Unterklasse von einer Oberklasse erbt.
Dabei erben die Unterklassen die Attribute und Methoden der Oberklassen.
Auch das Überschreiben von Methoden ist möglich. Also in Unterklassen Methoden zu definieren, die denselben Namen und dieselben Parameterlisten wie Methoden in der Oberklasse haben, um ihr Verhalten anzupassen. Dies macht man in Java indem man „@Override“ über die Methode schreibt.
Bsp in Java:
Durch „extends“ wird Java die Oberklasse festgelegt. Die Attribute werden durch „super“ vererbt, sodass sie dann auch bestimmt und in den Methoden benutz werden können. Es können aber auch zusätzlich Attribute definiert werden.
