Formel
Eine Formel ist eine Gleichung, die Berechnungen für die Daten in Ihrer Tabelle durchführt.
Formeln in Tabellenkalkulationsprogrammen kann man sich so ähnlich wie Formeln in Mathe vorstellen. Vor einer Formel muss immer ein Gleichheitszeichen stehen. Dann kann einfach mit den verschiedenen Rechenzeichen gearbeitet werden.
Funktion
Eine Funktion auch in Programmiersprachen ein datenverarbeitender Prozess. Also nimmt er Daten auf, die durch diese Funktion verarbeitet werden und am Ende bekommt man ein Resultat zurück. Sozusagen ist eine Funktion eine vordefinierte Formel, die spezifische Berechnungen durchführt.
Zellbezüge
Ein Zellbezug beschreibt die Zuordnung von Formelvariablen zu konkreten Werten in den entsprechenden Eingabefeldern (Zellen) in Tabellenkalkulationen.
Anstatt einer festen Zahl verweist wird mit einem Zellbezug auf eine andere Zelle verwiesen, um den Wert in dieser Zelle für die Funktion/Formel zu verwenden. Bei Zellbezügen unterscheidet man zwischen relativem und absolutem Zellbezug.
Dabei bezieht sich der absolute Zellbezug immer auf eine bestimmte Zelle. Dieser wird durch $-Zeichen vor den Buchstaben und die Zahl erzielt. Bsp: =SUM($A$1;$B$1)
Ein relativer Zellbezug, im Gegensatz berechnet Zellen immer relativ zu ihrer Position und bewegt sich so immer mit. Bsp: =SUM(A1;B1) dann wird eine Zelle darunter =SUM(A2;B2) stehen. Dies funktioniert sowohl horizontal als auch vertikal.
Diese zwei Arten von Zellbezügen können auch gemischt werden, sodass vertikal ein absoluter Zellbezug besteht und horizontal ein relativer oder umgekehrt. Dabei wird das $-Zeichen nur vor den Buchstaben oder die Zahl geschrieben.
Datenflussdiagramm
Ein Datenflussdiagramm stellt die Art der Verwendung, die Bereitstellung und Veränderung von Daten innerhalb eines Programms dar. Es kann auch dazu verwendet werden, den Datenfluss eines Prozesses oder einer Tätigkeit wiederzugeben.
Parameter
In der Informatik wird ein Parameter definiert als eine Variable in einer Funktion. Er dient dazu, bestimmte (oft veränderbare) Werte an diese Funktionen zu übergeben, um das Verhalten genau steuern zu können. Innerhalb einer Funktion wird dann mit den übergebenen Parametern gearbeitet.
